OVERVIEW

한봄 스튜디오는 웹에이전시로 웹사이트 및 모바일 앱 제작, 운영 관리를 주요 사업으로 진행하고 있습니다.

선정이유

퍼블리싱 공부를 하는 동안 자주 쓰지 못한 플러그인 및 제이쿼리 활용을 잘 할 수 있는 홈페이지를 찾는 도중 '한봄 스튜디오'를 발견하게 되었으며, 해당 홈페이지에 그동안 써보지 않은 효과들이 많아 선정하게 되었습니다.

PUBLISHING

1. HTML5

전체적인 컨텐츠를 Header, Section, Footer으로 나누어 작업

Section마다 주석으로 구분하여 쉽게 알아볼 수 있음

Section 안에 반복적으로 진행되는 컨텐츠는 ul,li로 작성

이미지 이용시 alt를 작성하여 접근성을 높임

class값을 기본으로 사용

2. CSS3

각 컨텐츠마다 주석으로 구분

동일한 태그의 css 값은 상단에 작성

간단한 애니메이션은 keyframes으로 작성

애니메이션을 만들 때 새로운 Class를 부여하여 제이쿼리로 완성

3. JQuery

wow.js 플러그인 사용

사용된 제이쿼리마다 주석으로 구분

addClass와 removeClass를 이용하여 애니메이션 완성

각 컨텐츠의 높이 값과 스크롤 값 등을 계산하여 애니메이션 완성

PUBLISHING

1. HTML5
  • 퍼블리싱의 기초가 되는 부분으로 전체적인 디자인을 보고 컨텐츠마다 큰 section을 나누어 만들었습니다.
  • section이 끝날때마다 주석으로 구분하여 알아보기 쉽고, 실수를 줄였습니다.
2.CSS3
  • css를 작성 할 때 컨텐츠마다 시작할 때 주석을 달아 수정 및 컨텐츠를 찾을 때 빠르게 확인할 수 있도록 하였습니다.
  • 태그들 중 동일하게 들어가는 css값은 상단에 작성하였습니다.
  • 간단한 움직임은 css에서 keyframes을 이용하여 작성하였습니다.
3. JQuery
  • wow.js 플러그인으로 전체적인 컨텐츠의 움직임을 제어 하였습니다.
  • 이 밖에 다른 효과들은 직접 제이쿼리를 작성하여 완성 하였습니다.
